1. Let, var and const
   Const is used when variables are meant to be constant
   let new variable declaration - block scoped within {}
   var old variable declaration - function scoped (anywhere in the function) [Not recomended]

2. Null and Undefined
   Null -> empty variable. Intetionally set a variable empty
   Undefined -> variable is declared but not defined

3. Undefined is a datatype but Null is an object
   typeof(undefined) -> undefind
   typeof(null) -> object

4. Equality check (==) and comparisions (<, >, <=, >=) works differently
   null == 0 => false. it checks value 
   null === 0 => false. it checks value and datatype
   null>=0 || null<=0 || null>0 || null<0 
   true       true       false     false
   Comparisions converts null into 0. Now do that math :)

5. == and === difference
   == => checks just value
   "2" == 2 => true. checks only value
   === => checks value and datatype
   "2" === 2 => false. datatype check
